AGraph - Features & Controls

Workflow

  • Analyze - Load selected keyframes from timeline into graph
  • Apply - Apply current graph curve to selected keyframes
  • Select 2 or more keyframes in timeline before analyzing
  • Midpoints are automatically generated when applying N-point curves to 2-point selections
  • Supports 1D properties (Opacity, Rotation, etc.) and multi-dimensional properties (Position, Scale)

Graph Interaction

  • Drag handle - Adjust easing curve shape
  • Shift + Drag handle - Constrain to horizontal or vertical
  • Alt/Option + Drag midpoint handle - Move one handle independently (break linking)
  • Double-click handle - Edit Speed & Influence numerically
  • Double-click midpoint - Edit Time & Value numerically
  • Shift + Click on curve - Add midpoint at clicked position
  • Alt/Option + Click midpoint - Delete midpoint

Graph Controls

  • Fit button - Reset graph scale to 100%
  • Scale +/- buttons - Zoom in/out the graph view
  • V button (green) - Toggle velocity graph overlay
  • A button (red) - Toggle acceleration graph overlay
  • Layout button - Switch between vertical/horizontal layout

Presets

  • Click preset card - Load preset curve into graph
  • Right-click preset card - Rename, move to group, or delete
  • Save button - Save current curve as new preset
  • + button - Create new preset group
  • Edit/Delete group buttons - Manage preset groups
  • Card size slider - Adjust preset card display size
